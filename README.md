# MoList 

MoList is a simple ToDo List app built using ReactJS. 

### Clone this repo 
Run the following commands in your terminal to clone this repo and customize it 

```
git clone https://github.com/Moise1/todo_react.git 
cd todo_react 
npm install 
npm start
``` 

Happy coding!  :)
